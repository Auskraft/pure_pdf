# Changelog

## 1.0 (MVP) — 2026
First release. A minimal, ad-free, fully-offline PDF reader.

- **Library** — recent documents (list / grid, density), thumbnails, size & date, empty state, "Открыть файл" via SAF.
- **Reader** — PdfiumAndroid rendering, vertical scroll, pinch-zoom + double-tap, rotation, page-jump dialog, slider, progress indicator.
- **Position memory** — remembers page + zoom per file; restores on reopen.
- **Search** — whole-document text search with highlights and next/previous.
- **Bookmarks** — add / remove / jump, stored locally.
- **Night mode** — dark scheme + smart-inverted "paper", persisted.
- **Appearance** — 4 accent presets (Material 3 from seed), library view, density.
- **Privacy** — no INTERNET permission, no analytics, no ads, fully offline.
- **Intents** — open via VIEW / SEND; registers as a PDF viewer.
- **App** — documentation (privacy / terms / licenses) and in-app rating (store + feedback, placeholders).
