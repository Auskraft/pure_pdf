# Pure PDF

Минималистичный Android-просмотрщик PDF: **без рекламы, без аккаунтов, полностью офлайн**. Одна задача, сделанная хорошо — быстро открыть и читать PDF.

## Возможности
- Открытие PDF из файлов (SAF) и через «Поделиться»; регистрация как просмотрщик PDF по умолчанию.
- Библиотека недавних документов (список/сетка, плотность), миниатюры, размер и дата.
- Читалка: вертикальная прокрутка, плавный рендеринг, pinch-to-zoom, двойной тап, поворот экрана.
- Запоминание позиции (страница + масштаб) для каждого файла.
- Переход по номеру страницы, индикатор прогресса, слайдер.
- Полнотекстовый поиск с подсветкой и переходом по совпадениям.
- Закладки (локально).
- Ночной режим с «умной» тёмной бумагой (не чёрно-белая инверсия).
- 4 акцентных пресета (Material 3 из seed-цвета), вид библиотеки, плотность — сохраняются между запусками.
- Документация (конфиденциальность / условия / лицензии) и оценка приложения.

## Приватность
Нет разрешения `INTERNET`, нет хранилищных разрешений, нет аналитики и рекламных SDK. Документы и данные (недавние, закладки, позиция, настройки) не покидают устройство.

## Стек
Kotlin · Jetpack Compose · Material 3 · Room · DataStore · [PdfiumAndroid](https://github.com/johngray1965/PdfiumAndroidKt) (рендер + текстовый слой) · [MaterialKolor](https://github.com/jordond/MaterialKolor) (схема из акцента).

Платформа: Android 10+ (minSdk 29, target/compile 35).

## Сборка
Откройте проект в Android Studio (он использует свой JBR) → Gradle Sync → ▶.

Из терминала нужен JDK в окружении:
```powershell
$env:JAVA_HOME = "A:\AS\jbr"   # или ваш JDK 17+
.\gradlew installDebug
```
Подробности об архитектуре и реализации — в [ARCHITECTURE.md](ARCHITECTURE.md). История изменений — в [CHANGELOG.md](CHANGELOG.md).

## Перед публикацией
