# Pure PDF

Минималистичный Android-просмотрщик PDF: **без рекламы, без аккаунтов, полностью офлайн**. Одна задача, сделанная хорошо — быстро открыть и читать PDF.

## Возможности
- Открытие PDF из файлов (SAF) и через «Поделиться»; регистрация как просмотрщик PDF по умолчанию.
- Библиотека недавних: список/сетка, плотность, **реальные превью первой страницы**, размер · число страниц · дата.
- Читалка: вертикальная прокрутка, плавный рендеринг, поворот экрана. **Два режима зума** — жестом (pinch с фокусом под пальцами) либо **стеклянными кнопками** (фото-пан одним пальцем); двойной тап.
- **Minimap-индикатор прокрутки** у правого края: перетаскиванием — быстрый переход, со стеклянным пузырьком страницы.
- **Плавающий навбар в стиле «жидкое стекло»** (блюр + полупрозрачность, Haze).
- Запоминание позиции (страница + масштаб); переход по номеру; полнотекстовый поиск с подсветкой; закладки.
- Ночной режим с «умной» тёмной бумагой (не чёрно-белая инверсия); 4 акцента (Material 3 из seed-цвета), вид библиотеки, плотность — сохраняются между запусками.
- Экран согласия при первом запуске; документация (соглашение / конфиденциальность / обработка ПД / лицензии); оценка приложения.

## Приватность
Нет разрешения `INTERNET`, нет хранилищных разрешений, нет аналитики и рекламных SDK. Документы и данные (недавние, закладки, позиция, настройки) не покидают устройство.

## Стек
Kotlin · Jetpack Compose · Material 3 · Room · DataStore · [PdfiumAndroid](https://github.com/johngray1965/PdfiumAndroidKt) (рендер + текстовый слой) · [MaterialKolor](https://github.com/jordond/MaterialKolor) (схема из акцента) · [Haze](https://github.com/chrisbanes/haze) (стеклянный блюр).

Платформа: Android 10+ (minSdk 29, target/compile 35). Релизный APK ~11 МБ (R8 + arm64-v8a/armeabi-v7a).

## Сборка и запуск
Открыть проект в Android Studio (он использует свой JBR) → Gradle Sync → ▶.

Из терминала нужен JDK 17+ в окружении:
```powershell
$env:JAVA_HOME = "A:\AS\jbr"   # или ваш JDK 17+
.\gradlew installDebug
```
